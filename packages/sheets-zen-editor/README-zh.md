## 简介

`@univerjs/sheets-zen-editor` 为 sheets 的每个 cell 提供了沉浸式编辑的体验。

## 使用指南

### 安装

```shell
# 使用 npm
npm install @univerjs/sheets-zen-editor

# 使用 pnpm
pnpm add @univerjs/sheets-zen-editor
```
